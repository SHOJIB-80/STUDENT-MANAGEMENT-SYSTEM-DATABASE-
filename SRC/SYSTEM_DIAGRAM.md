## Conflict Detection Algorithm

```
PSEUDO-CODE:

function checkScheduleConflicts(student_id, proposed_section_id):
    
    // Get proposed section's schedule
    proposed_schedule = get_schedules_for_section(proposed_section_id)
    
    // Get student's current sections
    current_sections = get_enrolled_sections(student_id)
    
    // For each currently enrolled section
    for each current_section in current_sections:
        current_schedule = get_schedules_for_section(current_section)
        
        // Compare schedules
        for each proposed_slot in proposed_schedule:
            for each current_slot in current_schedule:
                
                if proposed_slot.day == current_slot.day:
                    
                    if times_overlap(proposed_slot.time, current_slot.time):
                        
                        return CONFLICT_FOUND:
                            course_code: current_section.course_code
                            course_name: current_section.course_name
                            day: proposed_slot.day
                            time: current_slot.time_range
    
    return NO_CONFLICT

function times_overlap(time1_start, time1_end, time2_start, time2_end):
    // If one ends before other starts, no overlap
    if time1_end <= time2_start OR time2_end <= time1_start:
        return FALSE
    
    // Otherwise, they overlap
    return TRUE

EXAMPLE:
Proposed: Monday 08:00 - 09:30
Current:  Monday 09:00 - 10:30

times_overlap check:
- time1_end (09:30) > time2_start (09:00) âœ“
- time2_end (10:30) > time1_start (08:00) âœ“
â†’ OVERLAP DETECTED! Cannot enroll
```
