‚ùì FAQ

Q: Do I need to run the migration SQL?
A: Yes, run it once in phpMyAdmin before using the system. It creates necessary tables.
Q: Can I customize the time slots?
A: Yes! Edit START_HOUR, END_HOUR, COURSE_DURATION, and BREAK_DURATION in scheduling_helpers.php
Q: How does conflict detection work?
A: System checks if student already has a course at the same day/time. If yes, enrollment is blocked.
Q: Can a course have multiple sections?
A: Yes! Create multiple sections (A, B, C) for the same course with different times/instructors.
Q: What if I have other questions?
A: See README_SCHEDULING.md or SCHEDULING_SYSTEM_GUIDE.md for detailed answers.
